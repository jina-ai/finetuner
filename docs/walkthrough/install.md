# Install

![PyPI](https://img.shields.io/pypi/v/finetuner-client?color=%23ffffff&label=%20) is the latest version.

Make sure you have `Python 3.7+` installed on Linux/Mac/Windows:

````{tab} Basic install

```bash
pip install finetuner
```

````

````
