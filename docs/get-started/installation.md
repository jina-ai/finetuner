# Installation

![PyPI](https://img.shields.io/pypi/v/finetuner?color=%23ffffff&label=%20) is the latest version.

Make sure you have `Python 3.7+` installed on Linux/Mac/Windows:

```bash
pip install -U finetuner
```


Check your installation with:
```bash
pip show finetuner
```
